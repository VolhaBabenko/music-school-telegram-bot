# Telegram Bot для записи в музыкальную школу

Бот на Python + aiogram для онлайн-записи на уроки.

## Установка
1. `git clone https://github.com/ТВОЙ_USERNAME/music-school-telegram-bot`
2. `pip install -r requirements.txt`
3. Создай `.env` с `BOT_TOKEN=...`
4. `python main.py`

## Функции
- Выбор курса (пианино, гитара...)
- Запись на слот
- Админ-панель
